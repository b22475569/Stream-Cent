🚀 Windows 使用者 - 快速設定步驟：

【方法 1：雙擊啟動（推薦）】
1. 把所有檔案放在同一個資料夾
2. 雙擊 start_streamhub.vbs （無黑視窗，最乾淨）
   或
   雙擊 start_streamhub.bat （有視窗顯示狀態）

【方法 2：開機自動啟動】
1. 按 Win + R，輸入：shell:startup
2. 按 Enter 開啟「啟動」資料夾
3. 把 start_streamhub.vbs 的「捷徑」放進去
4. 重新開機後會自動啟動

【方法 3：桌面捷徑】
1. 在桌面右鍵 → 新增 → 捷徑
2. 瀏覽找到 start_streamhub.vbs
3. 命名為「Stream Hub」
4. 可以自訂圖示讓它更好看

🍎 Mac 使用者：

1. 打開「終端機」
2. 進入檔案所在資料夾：
   cd /你的檔案路徑
3. 執行：
   ./start_streamhub.sh

要設定開機自動啟動：
1. 系統偏好設定 → 使用者與群組 → 登入項目
2. 點 + 號，加入 start_streamhub.sh

💡 使用技巧：

✅ 優點：
- 點一下就啟動，完全自動化
- 自動開啟瀏覽器
- 解決 YouTube 錯誤 153 問題

⚠️ 注意事項：
- 首次使用前確保已安裝 Python
- 所有檔案必須放在同一個資料夾
- 如果 8000 port 被佔用，可修改腳本改用其他 port

🛑 關閉伺服器：
- 關閉瀏覽器不會停止伺服器
- Windows: 按 Ctrl+Alt+Delete → 工作管理員 → 結束 python.exe
- Mac/Linux: 終端機執行 killall python3

❓ 常見問題：

Q: 雙擊後沒反應？
A: 確認已安裝 Python，在命令列輸入 python --version 檢查

Q: 還是出現錯誤 153？
A: 確認使用的是 FIXED 版本，且透過 localhost:8000 開啟

Q: 想改成別的 port？
A: 編輯腳本，把所有 8000 改成你想要的數字（如 3000）

Q: 想開機自動啟動但不開瀏覽器？
A: 編輯 .vbs 或 .bat，把開啟瀏覽器那一行刪除或註解掉
